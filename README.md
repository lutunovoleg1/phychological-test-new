### Механика:
 В отличие от классического NCT-A, где ведут линии, в eNCT-A используется метод последовательных кликов/тапов по кругам с цифрами. При правильном нажатии круг меняет визуальное состояние (например, цвет).
### Погрешность и валидация:
 Исследование доказало, что eNCT-A имеет стопроцентную корреляцию с бумажной версией (PHES-батареей), но требует автоматической поправки на возраст. Платформа должна пресекать «мисклики», не останавливая общий таймер
### Стандарт электронного тестирования предписывает динамическую алгоритмическую генерацию нового поля для каждого сеанса со следующими правилами:
#### Ограничение перекрытий (Сетка): 
Экран виртуально делится на невидимую сетку (например, матрицу 5×5 или 6×6 ячеек). Внутри каждой ячейки может быть сгенерирован только один круг с цифрой. Это исключает наложение чисел друг на друга.
#### Случайное распределение (Randomization): 
Список чисел от 1 до 25 перемешивается случайным образом перед началом теста и распределяется по узлам сетки.
#### Безопасные отступы (Padding): 
Центр каждого круга должен иметь случайное смещение (девиацию) внутри своей ячейки сетки, но не выходить за ее границы. Это сохраняет ощущение хаотичности, как на оригинальном бумажном листе.
#### Масштабируемость: 
Координаты элементов должны рассчитываться в относительных единицах (например, процентах от ширины/высоты экрана), чтобы тест одинаково отображался как на мониторе компьютера, так и на планшете.

## Запуск проекта

### Требования

- Python 3.10 или новее;
- модуль `tkinter` и Tcl/Tk;
- дополнительные Python-библиотеки не требуются.

Проверить, доступен ли `tkinter`, можно командой:

```powershell
python -c "import tkinter; print('tkinter доступен')"
```

Если появляется ошибка `ModuleNotFoundError: No module named 'tkinter'`, установите Python с включенной опцией Tcl/Tk или используйте другой интерпретатор Python, где `tkinter` доступен.

### Команды запуска

Откройте терминал в папке проекта:

```powershell
git clone https://github.com/lutunovoleg1/phychological-test-new.git
cd C:\vs\phychological-test-new
```

Запустите приложение:

```powershell
python main.py
```

Если используете виртуальное окружение `.venv`, можно запустить так:

```powershell
.\.venv\Scripts\python.exe main.py
```

После запуска откроется окно приложения. Заполните ID/имя пациента, возраст и годы обучения, нажмите `Новый тест`, затем кликайте числа по порядку от `1` до `25`.

После завершения теста результат сохраняется в файл SQLite `nct_results.db` в корне проекта. Если файла еще нет, приложение создаст его автоматически.
